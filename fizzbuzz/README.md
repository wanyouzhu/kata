## Kata FizzBuzz:
You can find this kata here: http://codingdojo.org/kata/FizzBuzz/

### Tasks:

* Generate 100 turn numbers
    * () -> [1 ... 100] reports
* Tag generated turn numbers
    * 1 -> "1 1"
    * 3 -> "3 Fizz"
    * 5 -> "5 Buzz"
    * 15 -> "15 FizzBuzz"
    * 13 -> "13 Fizz"
    * 31 -> "31 Fizz"
    * 58 -> "58 Buzz"
    * 57 -> "57 FizzBuzz"
* Print the tagged turn numbers
    * [report(1), report(2), report(3)] -> "1 1\n2 2\n 3 Fizz\n"
