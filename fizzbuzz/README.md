## Kata FizzBuzz:
You can find this kata here: http://codingdojo.org/kata/FizzBuzz/

### Tasks:
* Generate 100 turn numbers
* Tag generated turn numbers
* Print the tagged turn numbers

### Tests:
* () -> 100 lines
* 1 -> "1"
* 3 -> "Fizz"
* 5 -> "Buzz"
* 15 -> "FizzBuzz"
* 13 -> "Fizz"
* 52 -> "Buzz"
* 35 -> "FizzBuzz"
