## Kata FizzBuzz:
You can find this kata here: http://codingdojo.org/kata/FizzBuzz/

### Tasks:

* Generate 100 turn numbers
    * () -> [1 ... 100] reports
* Tag generated turn numbers
    * 1 -> "1"
    * 2 -> "2"
    * 3 -> "Fizz"
    * 6 -> "Fizz"
    * 5 -> "Buzz"
    * 10 -> "Buzz"
    * 15 -> "FizzBuzz"
    * 45 -> "FizzBuzz"
    * 13 -> "Fizz"
    * 31 -> "Fizz"
    * 52 -> "Buzz"
    * 53 -> "Buzz"
    * 35 -> "FizzBuzz"
    * 53 -> "FizzBuzz"
    * 57 -> "FizzBuzz"
* Print the tagged turn numbers
    * [report(1)] -> "1 1"
    * [report(1), report(2)] -> "1 1\n2 2"
